## Run ✈️

- Clone the project

```bash
  git clone https://github.com/mark47B/Simple-React-Chat.git
```

- Go to the project directory

```bash
  cd Simple-React-Chat
```

- Install dependencies

```bash
  npm install
```

- Start the server

```bash
  npm start
```
